# Top One Algorithm

## python

    $ python3 top_one.py

    Exampel:
            Enter elemnts array: 7,1,2,5,8,8,8,5,5

            Output:
                    Number of Repetition: {7: 1, 1: 1, 2: 1, 5: 3, 8: 3}
                    Result: [5, 8]

## c++

    $ g++ top_one.cpp -o output
    $ ./output
    
    Example:
            Enter elemnts array(separate elements with commas): 7,1,2,5,8,8,8,5,5
            
            Output:
                Max Value Number of Repetition: 3
                Result:[5: 3]
                Result:[8: 3]
