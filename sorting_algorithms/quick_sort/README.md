# Quick Sort Algorithm

## python

    $ python3 quick_sort.py

    Example:
            
             Enter elemnts array: 4,21,42,5,56,7,1,4,2,65,4,2,1

             Output: [1, 1, 2, 2, 4, 4, 4, 5, 7, 21, 42, 56, 65]


## c++

    $ g++ quick_sort.cpp -o output
    $ ./output
    
    Example:
              Enter the array elements (separate elements with commas): 4,21,42,5,56,7,1,4,2,65,4,2,1 
              
              Output: 1 1 2 2 4 4 4 5 7 21 42 56 65
