# Selection Sort Algorithm

## python

    $ python3 selection_sort.py

    Example:
            
             Enter elemnts array: 20,34,4,21,76,89,0,12,1,2,99

             Output: [0, 1, 2, 4, 12, 20, 21, 34, 76, 89, 99]


## c++

    $ g++ selection_sort.cpp -o output
    $ ./output
    
    Example:
              Enter the array elements (separate elements with commas): 20,34,4,21,76,89,0,12,1,2,99 
              
              Output: 0 1 2 4 12 20 21 34 76 89 99
